Controle de Sinalização

Controle de Sinalização é uma plataforma para gestão e reporte de problemas de sinalização e infraestrutura urbana, permitindo que utilizadores registrem ocorrências em mapa interativo. Inclui painel administrativo para análise de solicitações e gestão de licitações de reparos.

GitHub

Sobre o projeto

A aplicação foi desenvolvida com React (Vite) no frontend e utiliza o Supabase para autenticação e armazenamento de dados. O sistema oferece funcionalidades completas para utilizadores comuns, como criação e acompanhamento de denúncias, além de um painel administrativo para análise, aprovação e gestão de solicitações e processos de licitação relacionados a reparos urbanos.

Funções do Sistema

  • Mapa Interativo: Visualização de problemas de sinalização e infraestrutura urbana em tempo real no mapa.
  • Registro de Denúncias: Criação de ocorrências com descrição, localização e detalhes do problema.
  • Acompanhamento de Solicitações: Consulta do status das denúncias realizadas pelo utilizador.
  • Autenticação de Utilizadores: Controle de acesso com login integrado ao Supabase.
  • Painel Administrativo: Visualização geral das ocorrências e gestão centralizada do sistema.
  • Gestão de Denúncias: Análise, aprovação ou recusa das solicitações enviadas.
  • Gestão de Licitações: Cadastro e controle de processos de reparo relacionados às ocorrências.
